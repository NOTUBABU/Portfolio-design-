# 🌐 Portfolio Website

A modern and responsive portfolio website built using **React.js** and **Vite**.
This project showcases my skills, projects, and achievements with a clean and interactive UI.

---

## 🚀 Features

* Responsive design (Mobile + Desktop)
* Smooth navigation
* Reusable React components
* Modern UI with Tailwind CSS
* Fast performance using Vite

---

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript / TypeScript
* Tailwind CSS

---

## 📂 Project Structure

* `src/` → Main source code
* `public/` → Static assets
* `components/` → Reusable components

---

## ▶️ How to Run Locally

```bash
npm install
npm run dev
```
